# React2_Shiny_Agency
Ce projet est un fil rouge dans le but de créer une application complète avec React.
Shiny_Agency utilise l'API pour pourvoir fonctionner, le projet combine en fait deux projets qui vont de paire.
Lancez d'abord l'API pour tourner le projet.
- INFOS: L'API tourne sur le port 8000, assurez-vous qu'il soit libre avant de lancer.

👉 N'oubliez pas d'installer les node_modules avec `yarn` pour tester le projet 👈

### J'utilise dans ce projet :
  - "styled components", une solution communément appelée **CSS in JS** pour créer le style de mon site
  - `ESLint` et `Prettier` pour gagner du temps en norme de code et détecter au mieux les sources d'erreurs (Ils sont utilisés en entreprise pour s'assurer que tout le monde fonctionne dans le même esprit et les mêmes normes de code; c'était l'ocasion pour moi de les tester et les ajouter à mes skills.)
